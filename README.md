# ScreenShader
Screensaver for macOS that displays GLSL shaders including the ones from Shadertoy and GLSLSandbox.
Supports HDR on HDR/EDR screens.<br>
Heavily based on https://github.com/patriciogonzalezvivo/glslViewer<br>
Adapted to macOS screensaver Cocoa API by Anastasiy Safari<br>

This is initial commit, it will display .frag files located in `Documents/shaders` folder, randomly on every run.
